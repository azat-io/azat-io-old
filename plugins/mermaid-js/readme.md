VuePress plugin for using Mermaid.

## Installation

```sh
npm install --save vuepress-plugin-mermaid-js
```

## Usage

Add plugin to your VuePress config:

```js
import { mermaidJsPlugin } from 'vuepress-plugin-mermaid-js'

export default {
  plugins: [mermaidJsPlugin()],
}
```

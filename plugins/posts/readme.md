VuePress plugin for blogging.

## Installation

```sh
npm install --save vuepress-plugin-posts
```

## Usage

Add plugin to your VuePress config:

```js
import { postsPlugin } from 'vuepress-plugin-posts'

export default {
  plugins: [postsPlugin()],
}
```
